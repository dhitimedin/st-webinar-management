== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/st-webinar-management` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= How do I get started? =
1. Go to 'Webinar Management' menu on the WordPress Admin Screen and create a new Post.
2. Fill the required details. You can add image and select speaker from the right block panel.
3. You may set category for this webinar by selecting it from webinar tab of the settings panel.
4. Save/Publish and then view the Preview.
